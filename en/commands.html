Server Commands

Complete list of commands available for players on Capitalyx

How to use commands

Commands allow you to interact with the server and access various features. Simply type the command in the game chat.

Tip: Use /help in-game to see a quick list of main commands.

General Commands

/help
Displays the list of main available commands.
/time
Displays the current server time.
/players
Shows the number of connected players.

Job Commands

/job
Displays information about your current job.
/quitjob
Quits your current job.
Warning: You will lose your job progression.

Economy Commands

/pay [id] [amount]
Gives cash to another player.

Parameters:

  • id - Player ID to pay
  • amount - Amount to give

Vehicle Commands

/vehicle
Opens the vehicle management menu.
/park
Parks your vehicle at the current location.

Communication Commands

/pm [message]
Whispers a message to very close players.
/discord
Shows the link to the official Discord server.

Help Commands

/report [message]
Reports a problem or player to administrators.
/rules
Shows a reminder of the server's main rules.
